# Magic8Ball


## How to play
1. Ask a yes or no question.
2. Wait for your fortune to be revealed.
3. If unsatisfied, replay.

## How to play
1. Fork the repo by clicking the fork logo the on top right <img src="../../images/fork.png" width="300" height="60">
2. Clone the repo `git clone git@github.com:AseanK/beginner-python-games.git`
3. Head magic8ball folder
4. Run the file using python command `magic8ball.py`
